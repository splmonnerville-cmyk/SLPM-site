Mesures de Sécurité Implémentées

  • X-Frame-Options: DENY – Empêche l'inclusion dans une iframe.
  • Content-Security-Policy (CSP) – Limite les sources des scripts, styles et images.
  • X-Content-Type-Options: nosniff – Empêche le MIME-sniffing.
  • Referrer-Policy: strict-origin-when-cross-origin – Limite les fuites de données via les références.
  • Cache-Control: no-store – Désactive le cache pour les pages sensibles.
  • Upgrade-Insecure-Requests – Force l'utilisation de HTTPS.

Contenu Sécurisé

Cette page est un exemple de contenu sécurisé. Tous les éléments externes sont chargés via HTTPS, et les scripts inline sont désactivés par défaut.
